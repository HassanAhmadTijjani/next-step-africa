# NextStep Africa

A free opportunity platform helping African students and young
professionals discover life-changing opportunities — scholarships,
fellowships, internships, jobs and announcements, all in one place.

## Live Site

Coming soon

## Built With

- React + Vite
- Tailwind CSS
- Supabase (database and authentication)
- Vercel (hosting)

## Features

- Browse latest opportunities
- View full opportunity details with apply link
- Admin dashboard to add and delete posts
- Protected admin login with Supabase authentication
- SEO optimized pages

## Pages

- Homepage — latest opportunities
- Single Post — full opportunity details
- About — mission and contact info
- Admin Login — protected admin access
- Admin Dashboard — manage all posts

## Getting Started

### 1. Clone the repository

git clone <https://github.com/hassanahmadtijjani/next-step-africa.git>

### 2. Install dependencies

npm install

### 3. Create a .env file

VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_KEY=your_supabase_anon_key

### 4. Run the development server

npm run dev
